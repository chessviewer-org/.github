# ChessVision

**Open-source tools for visualizing chess positions.**

We build software that turns chess notation into clear, high-quality images — fast, private, and free to use. Everything runs entirely client-side.

[Live App](https://chessvision.org) · [Report an Issue](https://github.com/chessvision-org/chess-vision/issues) · [Sponsor Us](#support-the-project)

---

## What We Build

### [chess-vision](https://github.com/chessvision-org/chess-vision)

A high-performance visualization engine. Reads FEN notation and renders print-quality images (PNG, JPEG, SVG) at resolutions up to 30,208px. Built with React and the HTML5 Canvas API. No backend, no tracking, no uploads.

Made for people who need accurate diagrams without standing up a server — coaches, authors, and developers.

[**Try it →**](https://chessvision.org) &nbsp;·&nbsp; [Source](https://github.com/chessvision-org/chess-vision)

## How We Work

- **Privacy first:** No analytics, no cookies, no telemetry.
- **Open source:** Released under [AGPL-3.0](https://github.com/chessvision-org/.github/blob/main/LICENSE). Read, fork, and build freely.
- **Quality enforced:** Conventional commits, zero-warning lint gates, and automated reviews.
